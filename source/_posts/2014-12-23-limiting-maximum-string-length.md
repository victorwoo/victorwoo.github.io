_适用于 PowerShell 所有版本_

要限制输出的文本不会过长，您可以使用类似这样的的逻辑来缩短超过指定长度的文本：

    if ($text.Length -gt $MaxLength)
    {
      $text.Substring(0,$MaxLength) + '...'
    }
    else
    {
      $text
    }
