_适用于 PowerShell 所有版本_

这行代码将输入一个数字列表并对它们随机排序：

```
Get-Random -InputObject 1, 2, 3, 5, 8, 13 -Count ([int]::MaxValue)
```

也可以用管道，不过速度更慢：

```
1, 2, 3, 5, 8, 13 | Sort-Object -Property { Get-Random }
```
