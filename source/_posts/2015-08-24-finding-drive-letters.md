以下是一个查找保留驱动器号的简单函数：

    #requires -Version 3

    function Get-DriveLetter
    {
        (Get-WmiObject -Class Win32_LogicalDisk).DeviceID
    }

要列出所有正在使用的驱动器号，请使用以下代码：

    PS> Get-DriveLetter
    C:
    D:
    Y:
    Z:

    PS>

要查看某个给定的驱动器号是否被保留，可以使用这段代码：

    PS> $letters = Get-DriveLetter

    PS> $letters -contains 'c:'
    True

    PS> $letters -contains 'f:'
    False

    PS>
