每个 Windows 日志文件都有一个注册的事件源列表。要找出哪个事件源注册到哪个事件日志，您可以直接查询 Windows 注册表。

这段代码将列出所有注册到“System”事件日志的事件源：

    $LogName = 'System'
    $path = "HKLM:\System\CurrentControlSet\services\eventlog\$LogName"
    Get-ChildItem -Path $path -Name
