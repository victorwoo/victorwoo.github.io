## Q: 使用 CSS 还是 ion- 指令
A: [Ion vs div directives - Ionic](http://forum.ionicframework.com/t/ion-vs-div-directives/7524)

CSS 优势：
- 性能
- DOM 干净

ion- 指令优势：
- 功能更强，例如 `ion-list` 的 `show-reorder` 等 API

结论：只是要样式，用 CSS；需要增强的 API 功能，用 ion- 指令。
