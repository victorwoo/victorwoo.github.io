由于相当多的用户似乎无法切换到新的、更安全的基于 REST 的 v3 PowerShell 模块来管理 Exchange，团队决定添加一种方法让客户重新启用 Remote PowerShell 以延长宽限期（至少适用于 Microsoft 云租户）。
