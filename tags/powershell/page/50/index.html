PowerShell 技能连载 - PowerShell 7

虽然在 PowerShell 7 之前，使用目前的版本也是可行的，但我们建议您关注新闻并阅读以下链接：

https://devblogs.microsoft.com/powershell/the-next-release-of-powershell-powershell-7/
